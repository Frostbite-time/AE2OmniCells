# 类型模糊卡

> 将 OMNI 存储元件的分区规则从“匹配具体资源”改为“匹配资源类型”。

<Row>
  <ItemImage id="type_fuzzy_card" scale="4" />
</Row>

## 用途

类型模糊卡可安装在本模组的 OMNI / 复杂 OMNI / 量子存储元件上，包括对应的便携版本与创造存储元件。

安装后，元件的分区配置会只检查资源所属的 AE 类型，而不再检查配置栏里的具体资源是否相同。

- 在分区配置中放入任意一个物品，代表允许或拒绝“物品”这一类型。
- 在分区配置中放入任意一种流体，代表允许或拒绝“流体”这一类型。
- 若配置了其他 AE 资源类型，也会按同样方式把该类型作为匹配目标。

## 白名单与黑名单

- 默认白名单：只允许与配置样本相同资源类型的内容进入。
- 安装反转卡后：改为黑名单，拒绝与配置样本相同资源类型的内容，其余类型可以进入。
- 分区配置为空时：类型模糊卡不会产生额外限制。
