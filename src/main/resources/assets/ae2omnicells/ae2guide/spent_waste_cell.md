# Spent Nuclear Components

A dedicated storage line for handling **Spent Nuclear Waste** and (optionally) other **radioactive chemicals**.

## Availability & Requirements
- These items can be obtained and function as intended **only** when **Applied Mekanistics** is installed.
- Without this dependency, the items on this page may still be creatable but will not serve their intended purpose.

## Usage & Behavior
- In the **default configuration**, standard **OMNI** cells **do not** accept *Spent Nuclear Waste*; therefore, unless you change the config, you can **only** store it using the **Spent Nuclear Storage Cell** shown on this page.
- If your config **disallows all radioactive chemicals** in standard OMNI cells, the **Spent Nuclear Storage Cell** will **allow** storing those radioactive chemicals (including Spent Nuclear Waste).

> **Tip**: The exact accepted substances, stacking behavior, and capacity depend on your configuration and the implementation version.
